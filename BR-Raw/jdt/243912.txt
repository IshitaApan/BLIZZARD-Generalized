IProject#delete(..) should suppress deltas from CharsetDeltaJob

3.4, see bug 243132 for how this causes random test failures in JDT/UI When you delete a project with a custom encoding, a CharsetDeltaJob is scheduled which reports deltas for the encoding changes 500ms later. In bug 243132, it's a Java project whose .settings/org.eclipse.core.resource.prefs contains "encoding/src=ISO-8859-1". This happens because the deletion of the prefs file ends up in this call chain: CharsetDeltaJob.addToQueue(CharsetDeltaJob$ICharsetListenerFilter) line: 60	 CharsetDeltaJob.charsetPreferencesChanged(IProject) line: 88	 CharsetManager.projectPreferencesChanged(IProject) line: 256	 ProjectPreferences.preferencesChanged(IProject) line: 212	 ProjectPreferences.deleted(IFolder) line: 137	 ProjectPreferences.deleted(IResource) line: 165	 Folder(Resource).deleteResource(boolean, MultiStatus) line: 772	 DeleteVisitor.delete(UnifiedTreeNode, boolean) line: 67	 DeleteVisitor.visit(UnifiedTreeNode) line: 150	 UnifiedTree.accept(IUnifiedTreeVisitor, int) line: 99	 FileSystemResourceManager.delete(IResource, int, IProgressMonitor) line: 228	 ResourceTree.internalDeleteFolder(IFolder, int, IProgressMonitor) line: 350	 ResourceTree.internalDeleteProject(IProject, int, IProgressMonitor) line: 385	 ResourceTree.standardDeleteProject(IProject, int, IProgressMonitor) line: 831	 Project(Resource).unprotectedDelete(ResourceTree, int, IProgressMonitor) line: 1720	 Project(Resource).delete(int, IProgressMonitor) line: 706	 Project.delete(boolean, IProgressMonitor) line: 317	 JavaProjectHelper$1.run(IProgressMonitor) line: 273	 Workspace.run(IWorkspaceRunnable, ISchedulingRule, int, IProgressMonitor) line: 1800	 Workspace.run(IWorkspaceRunnable, IProgressMonitor) line: 1782	 JavaProjectHelper.delete(IJavaElement) line: 288	 PackagesViewContentProviderTests2.tearDown() line: 470	 PackagesViewContentProviderTests2(TestCase).runBare() line: 136	 ... When the Job is executed 500ms later, the resource change listeners are notified of changes in a project that does not even exist any more. IProject#delete(..) should suppress these deltas.
