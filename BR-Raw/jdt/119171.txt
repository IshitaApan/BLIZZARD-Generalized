[quick fix]  "add unimplemented methods" creates imports hiding classes in same package

package bug2; import java.util.Properties; public interface Foo { public void blah(Properties props); } package bug2; public class Properties { public int getPropertyCount() { return 42;} } package bug2; // add unimplemented methods will import "java.util.Properties" public class Bug implements Foo { public int baz() { Properties props = new Properties(); return props.getPropertyCount(); } } A fix would be to not import that class but leave it referenced by its full class name.
