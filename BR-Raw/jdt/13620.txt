The Java Source Attachment Preference Page has text with large amount of white space

TThe Java Source Attachment Preference Page has a leabel that sits in the middle of the page with a huge amount of white space. It's verticalAlignment should be set to SWT.ALIGN_BEGINNING. STEPS 1) Load a project from the repository 2) Select a source folder 3) Open Properties 4) Select Java Source Attachment 5) You will get an info label in the middle of the dialog. It should be at the top like other ones.
