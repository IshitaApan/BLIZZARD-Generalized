[jar exporter] generated exporter scripts should use encoding preference

3.1 RC1 The scripts that are generated by the Javadoc export wizard (javadoc.xml) and the Jar exporter (xxx.jardesc) use a hardcoded text encoding (UTF-8) These scripts should honor the new text file encoding preference. Information about text encoding in xml documents can be found at http://www.w3.org/TR/xslt#output
