Stack overflow

Just before it crashed it looked like Eclipse was getting sick. An error dialog appeared on the screen that said something like 'Couldn't find class file on classpath'. The dialog looked like it was flickering, as if it was being redrawn rapidly. Although the GPF is a VM problem, it looks like the underlying cause was a stack overflow.
