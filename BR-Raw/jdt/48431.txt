Incorrect order for modifiers in generated testcases [JUnit]

New -> JUnit Test Case On second page of wizard select 'Create final method stubs' generated code looks like: final public void testXyz() { } ... but should be: public final void testXyz() { } (Recommended by the JLS - http://java.sun.com/docs/books/jls/second_edition/html/classes.doc.html - section 8.1.1, last paragraph)
