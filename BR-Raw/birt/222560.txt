Create stored procedure with output parameter. create or replace package birt_test_pkg is procedure testOutParam(o_Value out int); end; create or replace package body birt_test_pkg is procedure testOutParam(o_Value out int) is begin o_Value := 200; end; end; 2. Create new report and configure datasource. 3. Create stored procedure dataset with this query: {call birt_test_pkg.testOutParam(?)} 4. Add output parameter param1. 5.
