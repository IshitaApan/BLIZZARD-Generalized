Exception is thrown when category values are null and type of X axis is set to DateTime

Created attachment 77706 Check type error Build ID: I20070625-1500 Steps To Reproduce: Reproduce: 1. Create a bar chart and open chart builder dialog. 2. Switch to "Select Data" tab and set dataset, select category series and Y series columns. Make sure all value of category series are null. 3. Switch to "Format Chart" tab and select "X-axis" sheet, set type of X-axis to DateTime. Expected result: The category label in X-axis should display as "null" and chart can be preview. Actual result: The chart can't be preview and exception is thrown.
