BIDI3.3:HCG_Incorrect order of BiDi words in Edit Text Item Dialog

This is BiDi specific defect. Please use BiDi characters instead of capital Latin letters in examples below. Setup: ------ 1. Follow up the instructions provided at http://www.eclipse.org/birt/phoenix/build/ and copy itext-1.3.jar and iTextAsian.jar files into plugins/com.lowagie.itext_*/lib directory. 2. Set default OS locale to Hebrew 3. Run Eclipse without any command line parameters. Steps to reproduce the defect ------------------------------ 1. Create Report Project 2. Create empty report template 3. Add label with BiDi text "HELLO WORLD" to report template 4. Double click the text 5. Compare how BiDi text appear in the Report Designer and in the Edit Text Item Dialog Actual result: OLLEH DLROW Expected result: DLROW OLLEH BiDi text is correctly displayed in Report Designer but has incorrect layout in Edit Text Item Dialog.
