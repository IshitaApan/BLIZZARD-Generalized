[KeyBindings] StyledText gets key event when pressing Ctrl+O

I20040330 When pressing Ctrl+O in the Java editor a key event is sent to the StyledText widget: e= KeyEvent (id=831300) 	character= (int:15) 	data= null 	display= Display (id=830444) 	doit= true 	keyCode= 111 	source= StyledText (id=830484) 	stateMask= 262144 	time= 182622146 	widget= StyledText (id=830484) This immediately closes the Quick views (Outline, Hierarchy) and causes an NPE (see bug 56795). Worked in 3.0 M8. Not sure whether this caused by changes in Platform UI (key binding support) or SWT (BidiUtil seems to trigger the key event).
