[Preferences] Performance pref page is layer breaker and causes errors

I20050329-2000 The new Performance preference page directly reference JDT UI stuff but is located in org.eclipse.ui.ide. In addition to being a layer breaker this causes errors for users which do not have JDT installed (e.g. the Platform SDK drop) and it also shows a dialog saying that the page contains invalid values.
